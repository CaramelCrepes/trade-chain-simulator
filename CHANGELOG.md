# Changelog

All notable changes to TradeChain Simulator will be documented here.

## [1.0.0] - 2024-03-01

### Added
- Initial release
- SHA-256 blockchain simulation from scratch
- Three role-based dashboards — Exporter, Importer, Customs Officer
- Four trade document types — Bill of Lading, Certificate of Origin, Customs Declaration, Letter of Credit
- Live tamper detection and chain restoration demo
- Session-based login with role enforcement
- Shipment history tracking
- REST API endpoints for chain data and verification
- Persistent storage via ledger.json
- Demo seed script with realistic Mumbai → Hamburg shipment
